## allgemeine Elemente

- `div`: allgemeines Block-Element
- `span`: allgemeines Inline-Element

Block-Elemente: untereinander angeordnet, so breit wie möglich  
Inline-Elemente: nebeneinander angeordnet, so breit wie ihr Inhalt
