# Pseudoklassen

## Pseudoklassen

in CSS können mit `:` sogenannte Pseudoklassen abgefragt werden

## Pseudoklassen in Formularen

- `:checked`
- `:empty`
- `:valid`
- `:invalid`
- `:required`

## Pseudoklassen für Links, Buttons, ...

- `:hover`
- `:visited`
- `:active`

## Pseudoklassen für die Reihenfolge

- `:first-child`
- `:last-child`
- `:nth-child(2n)`
