# Plotting

## Plotting - advanced plots

- plotting data points with more than 2 features
  - advanced scatter plot (size, color)
  - scatter matrix
- plotting z = f(x, y)
  - contour plots
  - 3d plots
- plotting density of some distribution (advanced)
  - histogram
  - box plot
  - kde
  - violin plot
- plotting density (2d)
  - 2d histogram (hist2d, hexbin)
  - kde
