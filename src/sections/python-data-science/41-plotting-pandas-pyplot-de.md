# Plotting

## Graph

Pandas:

```py
series.plot()
dataframe.plot()
```

Matplotlib / Pyplot:

```py
plt.plot(data)
```

## Graph (x / y)

Matplotlib / Pyplot:

```py
plt.plot(x, y)
```

## Bar Chart

Pandas:

```py
df.plot.bar()
```

Matplotlib / Pyplot:

```py
plt.bar(x=[0, 1, 2, 3], height=y)
```

Seaborn

```py
sns.barplot(x=[0, 1, 2, 3], y=y)
```
