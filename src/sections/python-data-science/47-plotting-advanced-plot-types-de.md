# Erweiterte Arten von Plots

## Erweiterte Arten von Plots

- Datenpunkte mit mehr als 2 Features
  - Erweiterter Scatter Plot (Größe, Farbe)
  - Scatter Matrix
- Plotten von z = f(x, y)
  - Contour Plots
  - 3D Plots
- Dichte einer Verteilung
  - (Histogramm)
  - KDE
  - Violin Plot
- Dichte einer Verteilung (2D)
  - 2D Histogramm (hist2d, hexbin)
  - KDE
