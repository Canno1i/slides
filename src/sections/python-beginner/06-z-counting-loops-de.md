# Zählschleifen

## Zählschleifen

Folgendermaßen können wir in Python von 0 bis 9 zählen:

```py
for i in range(10):
    print(i)
```

Der Funktionsaufruf `range(n)` gibt die ersten `n` natürlichen Zahlen (beginnend bei 0) zurück

## Zählschleifen

Übung: Ausgabe einer Multiplikationstafel

```txt
1 x 7 = 7
2 x 7 = 14
3 x 7 = 21
4 x 7 = 28
...
```
