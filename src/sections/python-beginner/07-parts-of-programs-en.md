## constituent parts of programs

- programs
  - code blocks
    - statements
      - expressions

---

## statements across multiple lines

If a statement should encompass multiple lines it is usually written in parentheses:

```py
a = (2 + 3 + 4 + 5 + 6 +
     7 + 8 + 9 + 10)
```

Alternative: _escaping_ newlines with `\`

```py
a = 2 + 3 + 4 + 5 + 6 + \
    7 + 8 + 9 + 10
```
