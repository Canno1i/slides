# Set

## Set

Ungeordnete Menge von Elementen (ohne Duplikate)

```
primes = {2, 3, 5, 7, 11, 13, 17, 19}
```

## Set

Achtung: Ein leeres set erstellen wir immer mittels `set()`.

Warum ergibt der Ausdruck `{}` kein leeres set?

## Operationen auf Sets

```py
x = set('abc')
y = set('aeiou')
x | y
x & y
x - y
x <= y
```
