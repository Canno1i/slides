# Numbers

## int

integers of arbitrary size

## int

Other numeral systems:

```py
a = 42
b = 0o52
c = 0x2a
d = 0b101010
```

```py
e = int('101010', 2)
```

## float

64 bit integer

```py
a = 2.3
b = .2
c = 6e23
d = float('nan')
e = float('inf')
```

## complex

```py
a = 2 + 3j
```

## Operations with numbers

- Integer division: `10 // 3`
- Remainder: `10 % 3`
- Power: `2 ** 3`
