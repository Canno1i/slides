# Flexbox

## Display property

Möglichkeiten:

- `display: block`
- `display: inline`
- `display: none`
- `display: flex`

## Flexbox

Einfache Möglichkeit, Elemente _nebeneinander_ oder _untereinander_ anzuordnen
